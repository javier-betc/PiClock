Use this folder as an example of what optional services you could run on your machine that runs the webapp.
For me, this is a decomissioned "upcycled" consumer grade PC that I installed PVE on.
This is an Ubuntu VM for me that runs docker-compose-lamp.
This gives me the quick and easy way to run php, but it might ba bit "overkill" as I was planning to use mysql
and not csv files... but hey... you do you, feel free to perfect the whole setup imo.
The only service that I ended up wanting on my setup was to use the avahi capabilities to give the www enpoint
a "nickname" like e.g. http://piclock.local or just simple http://piclock works for me too...
At the same time it keeps your original "uvm" hostname endpoint alive, up to you if you want this or not...
